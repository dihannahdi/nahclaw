## Obsidian Integration Research
- **Date**: 2026-03-18 00:02 UTC
- **Task**: Research how to connect OpenClaw (running on VPS) with Obsidian (installed on personal laptop)
- **Findings**: OpenClaw has no built-in Obsidian integration. Several sync methods identified:
  1. **Git Sync**: Version-controlled, developer-friendly approach
  2. **Cloud Storage Sync**: Dropbox/Google Drive/Syncthing for real-time sync
  3. **SSH/SCP Sync**: Manual file transfer via SSH
  4. **Custom Webhook/API**: Advanced integration via custom skill
- **Recommendation**: Git Sync is most suitable for VPS + laptop setup
- **Key files to sync**: `MEMORY.md`, `AGENTS.md`, `memory/` folder, `skills/` directory
- **Security consideration**: Avoid syncing sensitive API keys/credentials

## Integration Methods Details
### Git Sync Workflow:
1. Initialize Git repo in `/root/.openclaw/workspace` on VPS
2. Push to GitHub/GitLab
3. Clone on laptop and open as Obsidian vault
4. Setup cron jobs for auto-commit/push from VPS

### Syncthing Alternative:
- Real-time bidirectional sync
- Install Syncthing on both VPS and laptop
- Share workspace folder between devices
